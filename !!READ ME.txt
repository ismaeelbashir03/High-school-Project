HOW TO PLAY:

INSTALL MYSQL WORKBENCH SO THE GAME CAN MAKE SAVES TO A LOCAL DATABASE

RUN MAIN.PY


INSTRUCTIONS

NORMAL LEVELS:

- WASD TO MOVE
- E TO ATTACK
- 1,2,3 FOR POWER CHANGES (U HAVE TO DEFEAT THE BOSSES TO UNLOCK)

BULLET HELL SHOOTER:
 - WASD TO MOVE 3D
 - MOUSE TO AIM 
 - LEFT MOUSE BUTTON TO SHOOT
